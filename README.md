# Movie Recommendation System

A Machine learning-based movie recommendation web application built using Flask. The system recommends similar movies based on genre similarity using Content-Based Filtering and Cosine Similarity.

## Technologies

### Frontend

- HTML5
- CSS3

### Backend

- Python
- Flask

### Machine Learning

- Pandas
- Scikit-learn
- CountVectorizer
- Cosine Similarity

### Dataset

- MovieLens Small Dataset
